## 1. Data We Actually Collect

We collect **only the minimum data necessary** for the Bot to function.

### Automatically Collected

::card-group
  ::card{icon="i-lucide-server"}
  **Server Data**
  
  - Server ID (Guild ID)
  - Channel IDs and Role IDs
  - Server configurations
  ::

  ::card{icon="i-lucide-user"}
  **User Data**
  
  - User ID (Discord snowflake)
  - Username and avatar (for display)
  - Discriminator (if applicable)
  ::

  ::card{icon="i-lucide-activity"}
  **Usage Data**
  
  - Command execution timestamps
  - Feature usage statistics (anonymized)
  - Error logs for debugging
  - Rate limit tracking
  ::
::

### Feature-Specific Data

::tabs
  :::tabs-item{label="Moderation" icon="i-lucide-shield"}
  - Warning logs (user ID, reason, timestamp, moderator ID)
  - Ban/kick records (user ID, reason, moderator ID)
  - Purge configurations and schedules
  - Auto-moderation settings
  
  ::note
  **Message content is never stored** except command arguments during execution.
  ::
  :::

  :::tabs-item{label="AFK System" icon="i-lucide-moon"}
  - User ID and server ID
  - AFK reason (max 100 characters)
  - Timestamp when set
  
  **Auto-deleted** when you return or manually clear.
  :::

  :::tabs-item{label="Counters" icon="i-lucide-hash"}
  - Counter type and configuration
  - Channel ID
  - Custom templates and goals
  - Role IDs (for role-based counters only)
  
  **No user-specific data** unless using role-based counters.
  :::

  :::tabs-item{label="Games" icon="i-lucide-gamepad-2"}
  **Guess the Number:**
  - User ID and server ID
  - Total/correct guesses
  - Current streak
  - Server statistics
  
  **No messages or personal info stored.**
  :::
::

## 4. Data Storage & Retention

### Where Data Lives

Data is stored on secure servers with:

::card-group
  ::card{icon="i-lucide-database"}
  **PostgreSQL Database**
  
  - Encrypted at rest
  - Regular backups (30-day retention)
  - Access-controlled
  - Geographically distributed
  ::

  ::card{icon="i-lucide-lock"}
  **Security Measures**
  
  - TLS/SSL encryption in transit
  - Multi-factor authentication
  - 24/7 automated monitoring
  - Regular security audits
  ::
::

### How Long We Keep It

| Data Type | Retention Period | Deletion Trigger |
|-----------|------------------|------------------|
| **Command logs** | 7 days | Automatic |
| **Error logs** | 30 days | Automatic |
| **AFK status** | Until removed | User returns or manual clear |
| **Counter configs** | Until deleted | Manual deletion or Bot removal |
| **Game statistics** | Until server removal | Bot kicked or data wipe requested |
| **Moderation logs** | 90 days (configurable) | Time-based or manual |
| **Inactive servers** | 90 days after removal | Automatic purge |

::warning
**No system is 100% secure.** While we implement industry-standard security, we cannot guarantee absolute security. Use at your own risk.
::

We share data **only** in these cases:

  We transmit data through Discord's API to execute Bot functions. This is necessary for operation and governed by [Discord's Privacy Policy](https://discord.com/privacy).
  :::

  :::accordion-item{label="Legal Obligations" icon="i-lucide-scale"}
  We may disclose data when required by law, court order, or to protect rights, safety, and security.
  :::

  :::accordion-item{label="Abuse Prevention" icon="i-lucide-shield-alert"}
  We may share abuse information with Discord's Trust & Safety team to maintain platform integrity.
  :::
::

### Third-Party APIs

Some features use external APIs with **no personal data shared**:

| Feature | Service | Data Shared |
|---------|---------|-------------|
| Cat images | The Cat API | None |
| Jokes | JokeAPI | None |
| Memes | Meme-API | None |
| GIFs | Giphy / waifu.pics | None |

We do **not** send user IDs, messages, or identifying information to these APIs.

## 6. User Control & Deletion

You have complete control over your data:

### Your Rights

::card-group

  **Right to Access**
  
  Request a copy of all data we hold about you or your server.
  
  **Response:** Within 30 days
  ::

  ::card{icon="i-lucide-edit"}
  **Right to Rectification**
  
  Correct inaccurate data using Bot commands or contact us.

  **Right to Erasure**
  
  Request deletion of your data from our systems.
  
  **Response:** Within 30 days
  
  **Note:** Some data may be retained for legal compliance

  ::card{icon="i-lucide-download"}

  **Response:** Within 30 days

::
