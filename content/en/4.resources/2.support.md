# Support & Community

## Getting Help

**Self-Help First:**
1. Check relevant section in this documentation
2. Read the Troubleshooting guide (fixes most issues)
3. Review Permissions guide (99% of issues are permissions)
4. Check FAQ (common questions answered)

**Still Stuck?**

**Support Server**
- Join our Discord community
- Ask questions in support channel
- Get real-time help from team and community
- [Invite Link: https://discord.gg/RZP3A9dq7S]

**What to Include When Asking for Help:**
1. What were you trying to do?
2. What error did you get? (paste full error)
3. Screenshot of your role hierarchy (for permission issues)
4. What you've already tried to fix it
5. Which server? (if relevant)

## Common Issues Quick Fixes

**Commands not working?**
- Move Lumière's role higher in hierarchy
- Ensure bot has required permissions
- Check Permissions guide

**Moderation actions failing?**
- Ensure Moderate Members permission enabled
- Verify bot role is above target user's role
- Try Administrator as temporary test

**Counting/Counters broken?**
- Reset with `/counting-reset` or `/counter-remove`
- Re-setup the feature
- Check bot has required permissions

**Voice channels not working?**
- Ensure it's a voice category, not text
- Verify bot has Manage Channels permission
- Try `/tempreset` and setup again

## Reporting Bugs

Found a bug? Report it on the support server.

**Include:**
- Exact steps to reproduce
- What you expected to happen
- What actually happened
- Error message (if any)
- Screenshots helpful

## Privacy

Lumière stores minimal data:
- Server settings (prefix, config)
- Moderation cases (for accountability)
- Counting statistics (game data)
- Nothing personally identifying

No selling data, no tracking, no advertising.

## Rate Limiting

Lumière respects Discord rate limits. If you execute many commands quickly:
- Some commands may queue
- No penalty, just delayed response
- Automatic retry built-in

Don't intentionally spam commands.
