## Guess the Number

Competitive number-guessing game with achievements and streaks.

### Quick Setup
```bash
/gtn set channel:#games        # Enable game
/gtn range min_number:1 max_number:100  # Change range
```

### How to Play
Type a number in the game channel. Get instant feedback:

### Commands

| Command | Description |
|---------|-------------|
| `/gtn stats` | Server statistics |
| `/gtn profile [@user]` | Personal stats & rank |
| `/gtn leaderboard [period]` | Top 20 players |
| `/gtn set reveal:True` | Reveal current number |
| `/gtn wipe` | Delete all data (Owner only) |

## Games

### Trivia
```bash
/trivia [category:Any] [difficulty:Any]
```
Answer questions via buttons or chat. **One attempt per user.**

**Features:**
- 13 categories (General, Science, Sports, etc.)
- Streak tracking with bonuses
- Daily first-win bonus (+100 pts)
- Scoring: Base + Speed + Streak bonuses

::note
Time limits: Easy (25s), Medium (30s), Hard (35s)
::

### Fast Type Challenge
```bash
/fasttype [difficulty:Easy/Medium/Hard/Insane]
```
Type phrases as fast as possible. Shows WPM and speed rating.

### Memory Game
```bash
/memorygame [difficulty:Easy/Medium/Hard/Insane]
```
Memorize emoji sequences (4-10 emojis). Score = Base + Speed bonus.

## FAQ

::accordion
  ::accordion-item{label="Do stats carry across servers?"}
  No. Each server has independent stats and leaderboards.
  ::
  
  ::accordion-item{label="Can I disable specific commands?"}
  Use Discord's command permissions in **Server Settings ‚Üí Integrations**.
  ::
  
  ::accordion-item{label="Why do GIFs sometimes fail?"}
  External API timeouts. Commands still work without GIFs.
  ::
  
  ::accordion-item{label="Math expressions in GTN?"}
  Supported: `+`, `-`, `*`, `/`, `%`, `**`
  Not supported: Functions like `sqrt()` or variables
  ::
::
