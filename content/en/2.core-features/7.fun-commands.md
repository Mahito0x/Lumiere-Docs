## 'Guess the Number' Game

This is a competitive number-guessing game with leaderboards, achievements, and streak tracking.

### Quick Start
```bash
# Setup (Admin/Moderator)
/gtn set channel:#games

# Play
Just type a number in the game channel!

# View Stats
/gtn stats
/gtn profile
/gtn leaderboard
```

### Setup Commands

::card-group
  ::card{icon="i-lucide-settings"}
  **Configure Game**
```bash
  /gtn set channel:#channel
```
  
  Sets up 'Guess the Number' in the specified channel with the default range (1-50).
  ::

  ::card{icon="i-lucide-sliders"}
  **Custom Range**
```bash
  /gtn range min_number:1 max_number:100
```
  
  Changes the number range and picks a new number.
  ::

  ::card{icon="i-lucide-toggle-left"}
  **Enable/Disable**
```bash
  /gtn set
```
  
  Toggles the game on or off without the channel parameter.
  ::

  ::card{icon="i-lucide-eye"}
  **Reveal Number**
```bash
  /gtn set reveal:True
```

  Shows the current number and starts a new game.

## Tips & Best Practices

### For Server Owners

## Troubleshooting

### Common Issues

::accordion
  ::accordion-item{label="'Guess the Number' is not responding to guesses."}
  **Causes:**
  - You are in the wrong channel (check `/gtn stats` for the correct game channel).
  - The game is disabled (a moderator may have toggled it off).
  - The 2-second cooldown is active (wait and try again).
  
  **Fix**: Verify that the game is enabled with `/gtn set` and that you are in the correct channel.
  ::

  ::accordion-item{label="Math expressions are not working."}
  **Supported operators**: `+`, `-`, `*`, `/`, `%`, `**` (power), `-` (negative)
  
  **Not supported**: Functions like `sqrt()` and `sin()`, or variables.
  
  **Examples**: âœ… `50/2`, `10+5*2` | âŒ `sqrt(25)`, `x+5`
  ::

  ::accordion-item{label="Action commands are showing an error."}
  **Self-action prevention**: You cannot use actions on yourself. Please choose another member.
  
  **Bot issues**: There may be temporary API failures. Please wait 30 seconds and try again.
  ::

  ::accordion-item{label="My achievements are not showing."}
  Achievements will only appear when you unlock them (the first time you meet the criteria). They are embedded in the win message, so check carefully!
  ::
::

## FAQ

::accordion
  ::accordion-item{label="Can multiple people guess at the same time?"}
  Yes! The game uses async locks to safely handle concurrent guesses. The first person to guess the correct number wins.
  ::tip
  However, it is best to turn on the cooldown.
  ::
  ::

  ::accordion-item{label="Do my 'Guess the Number' stats carry across servers?"}
  No. Each server has its own independent stats, leaderboards, and configurations.
  ::

  ::accordion-item{label="What happens to my streak if I make a wrong guess?"}
  Your streak will reset to 0 immediately after any incorrect guess. You can build it back up by guessing correctly!
  ::

  ::accordion-item{label="Why do some GIFs fail to load?"}
  External APIs occasionally time out or have rate limits. The command will still work, but there will be no GIF.
  ::

  ::accordion-item{label="Can I disable specific fun commands?"}
  Not natively. However, you can use Discord's command permissions in **Server Settings â†’ Integrations** to disable individual commands for each role or channel.
  ::
::
