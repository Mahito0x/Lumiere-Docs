## Essential Configuration

You don't need to set up everything at once. Start with the features that matter most to your server.

### 1. Logging System (Recommended First)

Know what's happening in your server. Track joins, leaves, message edits, and deletions.

```

```

**What you'll need:**
- A dedicated logging channel (create one called `#logs` or `#mod-logs`)
- Choose which events to track (the command will guide you)

- Add "Unverified" roles for servers with verification
